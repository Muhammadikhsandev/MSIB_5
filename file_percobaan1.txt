file ini adalah file yang dibuat didalam komputer yang kemudian akan diupload kedalam github
menggunakan perintah:
git add .-> untuk menambahkan file
git commit -m -> untuk menambahkan tanda atau komentar
git push -u origin master -> untuk memastikan file masuk kedalam repo
git config --global user.email "namaemail" -> settingan email kedalam git
git config --global user.name "username" -> untuk mamsukkan settingan username kedalam git
git init -> digunakan jika ingin komputer dengan github terhubung dan biasanya digunakan ketika akan melakukan remote
git config --list
git clone (salin code dari github)
cd folder/
git pull -u origin master --allow-unrelated-histories
